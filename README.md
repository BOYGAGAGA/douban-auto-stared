# 豆瓣自动标记

批量搜索豆瓣和资源站的工具，支持图片识别、标签页自动分组。

## 🚀 3秒上手

```
下载 → 解压 → 双击「启动.bat」→ 搞定！
```

就这么简单，不需要安装任何东西。

## ✨ 能做什么

| 功能 | 说明 |
|------|------|
| 🔍 批量搜索 | 输入多个关键词，一次全搜 |
| 📷 图片识别 | 截图粘贴，自动提取文字搜索 |
| 🌐 双站搜索 | 同时搜豆瓣 + 资源站 |
| 📑 标签分组 | 搜索结果自动归组（需装扩展） |
| 📝 搜索历史 | 记录搜索过的内容，一键重搜 |

## 📥 下载

点击 [Releases](../../releases/latest) 下载最新版，解压后直接用。

## 🔧 标签页分组（可选）

想让搜索结果自动分组？只需一步：

1. 解压 `extension.zip`
2. 打开 Edge → 地址栏输入 `edge://extensions/`
3. 开启「开发者模式」→ 点「加载已解口的扩展」→ 选 `extension` 文件夹

## 📖 怎么用

**搜索文字**
```
输入关键词 → 回车

多个关键词用逗号分开：
肖申克的救赎, 霸王别姬, 阿甘正传
```

**图片识别**
```
Ctrl+V 粘贴截图 → 自动识别 → 自动搜索
```

**设置（可选）**
```
点右上角 ⚙ → 配置腾讯云 OCR（更快更准，不配也能用）
```

## 🛡️ 隐私

- ✅ 所有数据存在你自己电脑上
- ✅ 不收集任何信息
- ✅ 图片在本地识别，不上传
