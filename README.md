# 豆瓣自动标记

批量搜索豆瓣和资源站的工具，支持图片识别、标签页自动分组。

## 🚀 3秒上手

**第1步：下载**

👉 [点我下载最新版](../../releases/latest/download/douban-auto-stared.zip)

**第2步：解压**

右键下载的 zip 文件 → 解压到当前文件夹

**第3步：打开**

双击文件夹里的「启动.bat」

搞定！浏览器会自动打开程序页面。

## ✨ 能做什么

| 功能 | 怎么用 |
|------|--------|
| 🔍 批量搜索 | 输入多个关键词，用逗号分开，回车 |
| 📷 图片识别 | Ctrl+V 粘贴截图，自动识别并搜索 |
| 🌐 双站搜索 | 同时搜豆瓣 + 资源站 |
| 📑 标签分组 | 搜索结果自动归组（需装扩展） |
| 📝 搜索历史 | 记录搜索过的内容，点击重搜 |

## 🔧 标签页分组（可选）

想让搜索结果自动分组？

1. [下载扩展](../../releases/latest/download/extension.zip) 并解压
2. 打开 Edge 浏览器，地址栏输入 `edge://extensions/`
3. 右下角开启「开发者模式」
4. 点「加载已解压的扩展」→ 选解压出来的 `extension` 文件夹

## 🛡️ 隐私

- ✅ 所有数据存在你自己电脑上
- ✅ 不收集任何信息
- ✅ 图片在本地识别，不上传
