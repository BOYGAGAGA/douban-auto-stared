# 豆瓣自动标记

一个简洁的豆瓣搜索工具，支持多关键词批量搜索、图片OCR识别、标签页自动分组。

## 功能

- **多关键词搜索** - 逗号/空格/换行分隔，批量打开搜索页
- **图片OCR识别** - 粘贴/拖拽/选择图片，自动提取文字并搜索
- **双站点搜索** - 同时搜索豆瓣和资源站
- **标签页分组** - Edge扩展自动将搜索标签页归为一组
- **历史记录** - 本地保存搜索历史，一键重搜

## 使用方法

1. 下载 [豆瓣自动标记.zip](../../releases/latest)
2. 解压后双击 `启动.bat` 打开程序
3. （可选）安装 `extension` 文件夹到 Edge 实现标签分组

## 技术栈

- 纯前端 HTML/CSS/JS，无需后端
- Tesseract.js 纯前端OCR（无需API key）
- 腾讯云OCR（可选，更快速准确）
- Edge Extension Manifest V3（标签分组）
