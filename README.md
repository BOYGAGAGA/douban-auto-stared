# 豆瓣自动标记

一个简洁的豆瓣搜索工具，支持多关键词批量搜索、图片OCR识别、标签页自动分组。

## ✨ 功能

- **多关键词搜索** - 逗号/空格/换行分隔，批量打开搜索页
- **图片OCR识别** - 粘贴/拖拽/选择图片，自动提取文字并搜索
- **双站点搜索** - 同时搜索豆瓣和资源站
- **标签页分组** - Edge扩展自动将搜索标签页归为一组（可选）
- **历史记录** - 本地保存搜索历史，一键重搜
- **隐私安全** - 所有数据存储在本地，不上传任何信息

## 📦 快速开始

1. 下载项目
2. 双击 `启动.bat` 打开程序（会用浏览器打开）
3. 开始搜索！

## 🔧 标签页分组（可选）

安装 Edge 扩展后，搜索多个关键词时标签页会自动归为一组：

1. 解压 `extension.zip`
2. 打开 Edge，地址栏输入 `edge://extensions/`
3. 开启左下角「开发者模式」
4. 点击「加载已解压的扩展」→ 选择解压后的 `extension` 文件夹

## 📖 使用方法

### 文字搜索
- 输入关键词（多个用逗号、空格或换行分隔）
- 点击搜索按钮或按回车

### 图片识别
- **Ctrl+V** 粘贴截图
- 拖拽图片到虚线框
- 点击「选择文件」上传图片

### 设置
- 点击右上角 ⚙ 按钮
- 可配置腾讯云 OCR API（可选，不配置也能用内置 OCR）

## 🛡️ 隐私说明

- 所有数据（搜索历史、API密钥）仅存储在浏览器本地
- 不收集任何个人信息
- 不上传任何数据到服务器
- 图片OCR在本地完成（使用Tesseract.js）
